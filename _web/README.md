# Shamo 🐔 / Frontend

## Local development

### How to run?

```zsh
npm run dev
```

### How to run with mock?

```zsh
npm run dev:mock
```

### How to run test?

```zsh
npm run test
```

### How to auto format files?

```zsh
npm run format
```

### How to update packages?

Uses [`npm-check-updates`](<https://github.com/raineorshine/npm-check-updates>)

```zsh
npm-check-updates -u
npm i
```
