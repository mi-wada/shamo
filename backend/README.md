## Local development

### Required tools

* [Docker](https://docs.docker.com)
* [Rust](https://www.rust-lang.org)
* [cargo-make](https://crates.io/crates/cargo-make)
* [cargo-watch](https://crates.io/crates/cargo-watch)

### How to run?

```zsh
cargo make run-local
```

### How to run test?

```zsh
cargo test
```

### How to auto format files?

```zsh
cargo fmt
```
